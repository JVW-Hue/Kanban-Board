# Simple Kanban Board

A beautiful, modern Kanban board for organizing your tasks with drag-and-drop functionality.

![Kanban Board](https://img.shields.io/badge/status-active-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue)

## What is it?

A Simple Kanban Board is a visual project management tool that helps you organize tasks into three columns:
- **To Do** - Tasks that need to be started
- **In Progress** - Tasks currently being worked on
- **Done** - Completed tasks

## Features

- Drag & drop cards between columns
- Add new tasks with title and description
- Delete tasks with one click
- Color-coded columns for easy visual identification
- Responsive design - works on desktop and mobile
- Auto-save to browser storage (persists after refresh)
- Beautiful modern UI with smooth animations

## Installation

### Option 1: Clone the Repository

```bash
git clone https://github.com/JVW-Hue/Kanban-Board.git
cd Kanban-Board
```

Then open `index.html` in your browser.

### Option 2: Download ZIP

1. Go to https://github.com/JVW-Hue/Kanban-Board
2. Click the green **Code** button
3. Select **Download ZIP**
4. Extract the ZIP file
5. Open `index.html` in your browser

## Usage

1. **Adding a Task**
   - Click the "+ Add Task" button in any column
   - Enter a task title (required)
   - Add a description (optional)
   - Click "Add" or press Enter

2. **Moving a Task**
   - Click and drag a task card
   - Drop it in the desired column

3. **Deleting a Task**
   - Hover over a task card
   - Click the × button in the top right corner
